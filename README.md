# Random-User-API-Performance-Test

## Project Summary:
This project has random user API performance test using JMETER

## Technology used:
- JMETER

## Pre requisites:
- Apache JMETER

## How to run
- ``` git clone ```
- ``` npm i ```
- ``` npi test ```

A report will be generate in your project source root
